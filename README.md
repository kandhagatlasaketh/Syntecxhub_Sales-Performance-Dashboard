Project Overview

*This project analyzes sales data to identify business trends, product performance, and regional sales patterns. The project includes data cleaning and exploratory data analysis using Python and an interactive dashboard built in Power BI.

##Objectives
Analyze sales performance across regions and categories.
Identify top-performing products.
Track sales and profit trends over time.
Build an interactive dashboard for business decision-making.
